
# Coronavirus

Este sito web busca exponer datos sobre la pandemia provocada por el virus [SARS-CoV-2](https://es.wikipedia.org/wiki/SARS-CoV-2), causante de la enfermedad [COVID-19](https://www.who.int/es/emergencies/diseases/novel-coronavirus-2019). El análisis de datos se hizo con el lenguaje de programación [R](https://www.r-project.org/).
Cada página de este sitio fue creada utilizando [R Markdown](https://rmarkdown.rstudio.com/).

A pesar de que existe un [sitio oficial del gobierno mexicano](https://coronavirus.gob.mx/) que trata sobre la pandemia, creo que es importante explicar qué nos muestran los datos sobre la evolución de la pandemia en nuestro país. Hay gráficas de las cuales es importante hablar y esto no se está haciendo.

Es mi responsabilidad como ciudadano con la formación y conocimiento técnicos producir estas gráficas así como explicarlas.

Finalmente, de la manera más atenta, por favor, [quédate en casa](https://www.youtube.com/watch?v=_TkmGO5ZZp4).

Los datos utilizados para la creación de este repositorio son los siguientes:

1. Dataset [_Our World in Data_](https://covid.ourworldindata.org/data/owid-covid-data.csv)
2. [John Hopkins University](https://systems.jhu.edu/research/public-health/ncov/)
